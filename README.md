# 🎮 OOP Pacman CLI Game (Java)

A simple Pacman-style console game developed in Java using Object-Oriented Programming (OOP) principles.

This project demonstrates class design, modular structure, and interaction between components in a CLI-based environment.

---

## 🚀 Features

- Player-controlled Pacman (W, A, S, D controls)
- Randomly moving enemy (ghost)
- Food collection system with score tracking
- Maze-based movement with walls
- Collision detection (game over condition)
- Console-based rendering

---

## 🧠 OOP Concepts Used

- **Encapsulation** – Each class manages its own data and behavior  
- **Abstraction** – Game logic is handled through methods  
- **Modularity** – Separate classes for Player, Enemy, and Board  
- **Reusability** – Code structure can be reused in other projects  

---

## 📁 Project Structure

PacmanGame.java   // Main game engine  
Pacman.java       // Player class  
Enemy.java        // Ghost class  
Board.java        // Game environment

---

## 🔄 Game Flow

- Board is displayed  
- Player enters movement input  
- Pacman moves if the path is valid  
- Food is collected and score increases  
- Enemy moves randomly  
- Collision is checked  
- Game ends if the enemy catches Pacman  

---

## ⚠️ Limitations

- Console-based only (no GUI)  
- Enemy uses random movement (no AI)  
- Fixed board size (10×10)  
- Single-level gameplay
